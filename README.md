# DisenoMovil
app/src/main/java/com/joel/diseomovil/MainActivity.kt
Pantalla de login creada en Android Studio

# Descripcion
Este proyecto muestra una pantalla simple con:
- Imagen principal
- Texto de bienvenida
- Boton Login y SignUp
- Seccion de redes sociales

Se usaron componentes basicos como Column, Row, Box, Text, Image y Modifier.

# Tecnologias
- Kotlin
- Android Studio
